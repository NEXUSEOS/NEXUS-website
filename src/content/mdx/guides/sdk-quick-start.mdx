# SDK Quick Start

Install NEXUS SDK packages from the Download Center and authenticate with your API key.

<MdxCallout>
  Package implementations ship from **nexus-sdk**. This guide documents the public contract.
</MdxCallout>

## Installation

```bash
npm install @nexus/sdk-core @nexus/sdk-behavior
```

## Authentication

Use `nexus login` or configure `NEXUS_API_KEY` for cloud API access.
